Contribute to Stencila

Stencila is an open-source, community-driven project. We encourage and welcome contributions from all community members. There are a number of ways you can contribute.

Develop Stencila

If you are a Javascript developer, you may be interested in helping out with: * [Stencila user interfaces](https://github.com/stencila/stencila) * [Stencila for Node.js](https://github.com/stencila/node) * [Stencila Desktop](https://github.com/stencila/desktop) * [Stencila CLI (Command Line Interface)](https://github.com/stencila/cli) If you develop code for data manipulation (in Python, R, and other languages), you may be interested in: * [Core library][libcore-contribute] * [Domain-specific libraries][libraries-contribute] If you are a Python developer, you may be interested in helping out with: * [Stencila for Python](https://github.com/stencila/py) * [Stencila Hub](https://github.com/stencila/hub) If you are a R developer, you may be interested in helping out with: * [Stencila for R](https://github.com/stencila/r) If you are into cloud computing, Docker and Kubernetes, you may be interested in: * [Stencila Cloud](https://github.com/stencila/cloud) If you build Docker containers, contribute to: * [Stencila Images](https://github.com/stencila/images)
